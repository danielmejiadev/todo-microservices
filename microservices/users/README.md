# Todo-Users-Microservice

## Description
A Microservice to manage the users resources.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# The microservice should be run on 127.0.0.1:6000 over TCP
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
